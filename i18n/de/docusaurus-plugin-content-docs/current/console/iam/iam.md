# Identity Access Management

Das IAM (Identity Access Management) Modul der Cloud Temple Konsole bietet eine umfassende Lösung für die Verwaltung von Identitäten und Zugängen.
Es ermöglicht die Verwaltung lokaler Konten und unterstützt gleichzeitig eine Hybridisierung mit externen Konfigurationen, die mit den gängigen Marktprotokollen wie OIDC und SAML kompatibel sind, um die Flexibilitäts- und Sicherheitsanforderungen moderner Umgebungen zu erfüllen.
