Cloud Temple bietet ein Private Networks-Angebot, das auf die Anforderungen von Unternehmen an sichere, leistungsstarke und flexible Connectivity abgestimmt ist. Aufbauend auf einer innovativen Architektur mit der VPLS-Technologie bietet diese Lösung ein privates Layer-2-Netzwerk, das transparent und skalierbar über alle Verfügbarkeitszonen hinweg ist.
