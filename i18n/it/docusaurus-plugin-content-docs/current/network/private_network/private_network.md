Cloud Temple offre un servizio di Reti Private progettato per soddisfare i bisogni delle aziende in termini di connettività sicura, performante e flessibile. Basato su un'architettura innovativa che utilizza la tecnologia VPLS, questa soluzione fornisce una rete privata di livello 2, trasparente ed estensibile attraverso tutte le zone di disponibilità.
